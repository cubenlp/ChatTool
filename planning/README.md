# Planning Workspace

更新时间：2026-03-08
状态：Draft（持续演进）

这个目录用于“边聊边写”，优先发散，不要求一次定型。

## 文件分工

- `guide.md`：开发指引（把想法转成可执行开发规则）。
- `roadmap.md`：方向路线图（顺序 + 工作量/难度，不绑定时间线）。
- `brainstorm.md`：脑洞池，先收集想法，不急着判断对错。
- `decision_log.md`：阶段性结论与取舍依据。
- `../planning.md`：对外主规划文档（相对稳定版本）。

## 协作约定

- 先记录：任何新想法先入 `brainstorm.md`。
- 再归并：相似想法聚类，提炼成候选方案。
- 后定案：有证据后再写入 `decision_log.md`，并同步 `planning.md`。
