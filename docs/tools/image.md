# AI 绘图工具推荐与配置指南

本文档整理了目前主流的免费/高性价比 AI 绘图工具：**通义万相**、**即梦 (Dreamina)**、**哩布哩布 (LiblibAI)**、**Bing Image Creator** 和 **Hugging Face**。包含了详细的工具介绍、免费额度说明以及配置使用流程。

---

## 1. 通义万相 (Tongyi Wanxiang)

阿里巴巴出品的大模型，依托阿里云的技术底座，主打企业级品质和多模态理解。

### 核心功能
*   **文生图/图生图**：支持多种艺术风格（油画、水彩、3D、写实等）。
*   **虚拟模特/试衣**：为电商设计提供模特替换和服装展示功能。
*   **涂鸦作画**：将简单的草图转化为高质量插画。
*   **背景生成**：自动为产品图更换适配的背景环境。

### 免费额度
*   **每日赠送**：每日登录通常赠送 **50 算力点**（约可生成 50 张图）。
*   **新用户奖励**：首次注册往往有额外的启动点数。

### 配置与使用流程
1.  **访问官网**：前往 [通义万相官网](https://wanxiang.aliyun.com/)。
2.  **账号登录**：使用阿里云账号、淘宝、支付宝或手机号登录。
3.  **开始创作**：
    *   在功能区选择“文生图”或“图生图”。
    *   输入中文或英文提示词。建议采用结构：`主体 + 动作 + 场景 + 风格描述`。
    *   **配置参数**：
        *   **比例**：选择 1:1, 4:3, 3:4, 16:9, 9:16 等。
        *   **风格**：选择预设的摄影、二次元、国风等滤镜。
    *   点击“生成”即可。

## 2. 即梦 (Dreamina)

抖音（字节跳动）推出的创作平台，在 AI 视频生成和高质量图像方面表现出色，是国内对标 Sora/Runway 的领军产品。

### 核心功能
*   **高质量图生影**：支持将生成的静态图一键转化为流畅的短视频。
*   **文生视频**：强大的动态理解能力，支持复杂指令的视频生成。
*   **智能画布**：支持图层化编辑、局部重绘、扩图。
*   **模型优势**：采用字节自研模型，在中文语义理解和光影质感上极强。

### 免费额度
*   **每日赠送**：每日赠送约 **60-100 积分**（普通生成每次消耗 1-2 积分）。
*   **限时活动**：经常有参与社区任务增加积分的活动。

### 配置与使用流程
1.  **访问官网**：前往 [即梦官网](https://dreamina.aicp.com/) 或下载移动端 App。
2.  **账号登录**：使用抖音账号或手机号一键登录。
3.  **开始创作**：
    *   进入创作界面，选择“图片生成”或“视频生成”。
    *   输入描述语，选择模型版本（如 V2.0 或 V3.0）。
    *   **配置参数**：
        *   **视频控制**：可配置运镜方向（推拉、平移、旋转）和运动强度。
        *   **智能参考**：支持上传参考图（控制姿势、边缘、深度等）。
        *   **分辨率**：支持最高 2K 级高清放大。

## 3. 哩布哩布 (LiblibAI)

国内最大的 Stable Diffusion (SD) 模型社区和创作平台，被誉为“中国版 Civitai”。适合进阶用户和专业设计师。

### 核心功能
*   **海量模型库**：拥有海量第三方 LoRA、Checkpoint 模型，涵盖各种细分风格。
*   **在线工作流**：支持在线运行 ComfyUI 和 WebUI，无需高配置显卡。
*   **LoRA 训练**：支持用户在线上传素材训练自己的风格模型。
*   **深度控制**：支持 ControlNet、IP-Adapter 等深度控制工具。

### 免费额度
*   **每日赠送**：每日登录赠送 **100 积分**（足以支持约 100 次基础生图）。
*   **免费模型**：绝大多数社区模型可免费在线调用或下载。

### 配置与使用流程
1.  **访问官网**：前往 [LiblibAI 官网](https://www.liblib.art/)。
2.  **账号登录**：手机号注册登录。
3.  **寻找模型**：
    *   搜索感兴趣的模型（如“写实摄影”、“赛博朋克”）。
    *   点击模型详情页的“在线生图”或“一键同款”。
4.  **开始创作 (在线生图界面)**：
    *   **基础配置**：输入正向/反向提示词。
    *   **参数调节**：
        *   **采样器 (Sampler)**：选择 Euler a, DPM++ 等。
        *   **步数 (Steps)**：通常设置为 20-30。
        *   **CFG 缩放**：控制提示词相关性，通常 7-9。
    *   **LoRA 权重**：在右侧栏添加 LoRA 模型并调整权重 (0.1 - 1.0)。
    *   **高级模式**：如需更复杂操作，可切换到 ComfyUI 模式搭建工作流。

## 4. Bing Image Creator (Microsoft Designer)

由 OpenAI 的 **DALL-E 3** 模型驱动，以其极强的语义理解能力和对中文指令的良好支持而闻名。

### 核心功能
*   **自然语言理解**：精准识别复杂的提示词，包括光影、构图及特定艺术风格。
*   **文字渲染**：相比旧模型，DALL-E 3 在图像中生成正确拼写的文字能力显著增强。
*   **多端整合**：可在 Bing 搜索、Edge 侧边栏及 Windows Copilot 中直接调用。

### 免费额度
*   **增强值 (Boosts)**：初始通常拥有 100 个“增强值”。每生成一组图片（通常为 4 张）消耗 1 个增强值。
*   **无限慢速生成**：增强值耗尽后，依然可以生成图片，但生成速度会变慢。
*   **自动恢复**：增强值通常每周自动补足。

### 配置与使用流程
1.  **访问官网**：前往 [Bing Designer](https://www.bing.com/images/create)。
2.  **账号登录**：使用 Microsoft 个人账号（Outlook/Hotmail）登录。
3.  **开始创作**：
    *   在搜索框输入描述词。支持中文，但使用英文描述通常效果更细腻。
    *   点击“创建”，等待几秒后即可查看 4 张候选图。
    *   点击图片可下载高分辨率原图。

## 5. Hugging Face (AI 绘图)

全球最大的开源模型社区，提供包括 Stable Diffusion、Flux.1 和 SDXL 在内的数千种绘图模型。

### 核心功能
*   **模型多样性**：不仅有官方基础模型，还有大量社区微调模型（如动漫风格、写实风格）。
*   **在线运行 (Spaces)**：无需本地显卡，直接在浏览器中运行由他人部署好的 Gradio 或 Streamlit 应用。
*   **API 接入**：开发者可以通过 Inference API 将绘图功能集成到自己的程序中。

### 免费额度
*   **Spaces 使用**：大部分公共 Spaces 是免费的，由 Hugging Face 提供基础算力。
*   **API 免费层**：提供免费调用额度，适合测试和小规模使用。

### 配置与使用流程
1.  **访问官网**：前往 [Hugging Face Spaces](https://huggingface.co/spaces)。
2.  **寻找模型**：
    *   搜索 "Text-to-Image" 或具体模型名（如 "Flux", "Stable Diffusion"）。
    *   选择一个热门的 Space（例如 [Flux.1-schnell](https://huggingface.co/spaces/black-forest-labs/FLUX.1-schnell)）。
3.  **开始创作**：
    *   **输入提示词**：在 Prompt 框中输入英文描述。
    *   **参数调节** (可选)：
        *   **Seed**：控制随机性。
        *   **Guidance Scale**：控制图像对提示词的忠实度。
        *   **Step**：数值越高细节通常越多。
    *   点击 "Run" 或 "Generate"。

## 开发者调用指南

本项目已集成上述部分工具的 API 调用。

### 1. 配置环境变量

首先，使用 `chatenv` 命令初始化配置：

```bash
# 查看支持的配置类型
chatenv init --help

# 交互式设置 Key (支持 tongyi, hf, liblib, bing 等)
chatenv init -i -t tongyi -t hf
```

或者直接编辑 `.env` 文件，添加以下变量：
*   **通义万相**: `DASHSCOPE_API_KEY`
*   **Hugging Face**: `HUGGINGFACE_HUB_TOKEN`
*   **LiblibAI**: `LIBLIB_ACCESS_KEY`, `LIBLIB_SECRET_KEY`
*   **Bing**: `BING_COOKIE_U`

### 2. 命令行使用

可以使用 CLI 直接生成图片：

```bash
# 使用通义万相生成
chattool client image generate -p tongyi "一只在屋顶晒太阳的赛博朋克猫" -o cat_tongyi.png

# 使用 Hugging Face 生成 (默认 FLUX.1)
chattool client image generate -p huggingface "A futuristic city at night, neon lights" -o city_hf.png

# 使用 Bing 生成
chattool client image generate -p bing "An astronaut riding a horse on Mars"
```

### 3. 代码调用

在 Python 代码中调用：

```python
from chattool.tools.image import create_generator

# 初始化生成器 (自动读取环境变量)
generator = create_generator("tongyi") 
# 或者手动传入 key: create_generator("tongyi", api_key="sk-...")

# 生成图片
result = generator.generate("一只可爱的猫")
print(result)
```
