# Workspace 初始化清单

1. **Discover** — 检查 workspace 和关联项目：看目录结构、现有工具、配置文件和 README。
2. **Ask** — 在写任何内容前，先向人确认关键问题：
   - 项目名称和目标是什么？
   - 当前可用工具有哪些，分别在哪里？
   - 当前最优先的事情是什么？
   - 有没有需要保留的现有状态或历史工作？
3. **Adapt** — 把 `AGENTS.md`、`MEMORY.md`、`thoughts/current.md` 改成项目相关的具体内容，替换掉所有占位文本。
4. **Initialise** — 在 `knowledge/memory/status/YYYY-MM-DD-status.md` 里写下项目初始状态。
5. **Create first task lanes** — 创建 `reports/<task-name>/` 和 `playgrounds/<task-name>/`，并把第一个具体任务写进 `reports/<task-name>/TASK.md`。
6. **Done** — 在本文件末尾追加 `completed: YYYY-MM-DD`。
